SyncSelection - Notepad++ Python Script

Синхронизация выделения текста между левой (View 0) и правой (View 1) панелями.
Поддерживает Notepad++ x64 и Python Script 3.0+ (Python 3.14).
Реализована защита от утечки колбэков при ручных перезапусках скрипта.

Вайбкодинг от Gemini 3.5 Flash

Скрипт запускался на 
Notepad++: 8.9.6.2 (portable) x64
Версия плагина: Python Script 3.0.25 (pre-release на базе Python 3.14)
