**SyncSelection** - Notepad++ Python Script

Synchronizes text selection between left (View 0) and right (View 1) panes.<br>
Works with different files opened in different panels.

Supports Notepad++ x64 and Python Script 3.0+ (Python 3.14).<br>
Implemented protection against callback leaks during manual script restarts.

*Vibe coding by Gemini 3.5 Flash*

The script was run on<br>
Notepad++: 8.9.6.2 (portable) x64<br>
Plugin version: Python Script 3.0.25 (pre-release based on Python 3.14)
